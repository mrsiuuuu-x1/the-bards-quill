# The Bard's Quill 

A browser extension that transforms boring email drafts into dramatic Shakespearean soliloquies.

## How it Works
1. Click the floating "Scroll" button in the corner of your screen.
2. Watch as your modern text is translated into Old English.
3. "Hello boss" becomes "Hark, my liege!"

## How to Install 
1. Clone this repository.
2. Open Chrome and go to `chrome://extensions`.
3. Enable **Developer Mode**.
4. Click **Load Unpacked** and select this folder.
